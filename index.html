Chave da API Anthropic (Claude)
Necessária para o agente IA responder no WhatsApp e no assistente jurídico
⚠ Não configurada
Obtenha sua chave em console.anthropic.com/settings/keys.
Com uma chave paga o agente responde sem interrupção. Planos a partir de U$5/mês — recarregue conforme o uso.
A chave fica salva localmente no seu navegador (localStorage) e nunca é enviada a terceiros.
📋 Como obter a chave paga:
  1. Acesse console.anthropic.com e crie uma conta
  2. Vá em Billing e adicione um cartão de crédito (cobrado por uso)
  3. Acesse API Keys → Create Key
  4. Cole a chave acima e clique em Salvar
